# android_demo
## 装备管理

基于Android的装备数据管理及数据同步。

功能包括：用户登录、装备列表管理(刷新、加载更多)、二维码扫描、自定义查询、设备详情展示、设备教程展示、图片查看、视频播放、FTP文件上传下载、远程数据同步等功能。
